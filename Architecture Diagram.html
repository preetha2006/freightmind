Layer 3 — Business Logic Services
Core Service Modules
⚙
scorer.py
Builds 12-feature vector per carrier. Calls model.predict(). Computes price_ratio vs market_rate. Applies priority and vehicle type encoding. Returns ranked list with composite scores.
Feature Eng.RankingNormalisation
💡
explainer.py
Uses SHAP TreeExplainer on the loaded XGBoost model. Generates per-prediction SHAP values. Maps values to human-readable impact labels (positive / negative / neutral).
SHAPTreeExplainerXAI
📡
Negotiation Signal Engine
Computes bid_price / market_rate ratio. If ratio exceeds threshold (1.10), flags negotiation signal, calculates savings in INR, and recommends target price.
Threshold LogicINR Calc
⟁
Award Simulation Engine
Iterates split_pct from 10 to 90 in steps of 10. For each split, computes weighted blended_score, blended_ontime_pct, expected_cost_inr between two carriers. Returns full range array.
Weighted AvgRange Sweep

ML Model Layer

XGBoost Regressor
Trained on 5,000 historical shipments. Target variable: carrier_score (0–100). 80/20 train-test split. Serialised to model.pkl via joblib. Auto-loaded at server startup.
